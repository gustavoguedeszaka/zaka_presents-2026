# ZAKA®

Tecnologia avançada com foco humano.

Plataforma institucional premium da ZAKA com páginas cinematográficas para produtos de inteligência artificial, visão computacional e inteligência estratégica.

## Stack Utilizada

- HTML5
- CSS3
- Vanilla JavaScript
- GSAP / Animações Customizadas
- Responsive cinematic UI

## Organização do Projeto

A estrutura segue o padrão estabelecido:
- `/assets`: Imagens, vídeos e arquivos de mídia.
- `/css`: Folhas de estilo da plataforma.
- `/js`: Scripts e lógica de interface.
- `/`: Páginas HTML dos produtos.
